This release contains one fix from 8.0.11. For information about new features in the 8.0 major release, see Section E.268.

E.256.1. Migration to Version 8.0.12

A dump/restore is not required for those running 8.0.X. However, if you are upgrading from a version earlier than 8.0.6, see Section E.262.

E.256.2. Changes

  • Remove overly-restrictive check for type length in constraints and functional indexes(Tom)
